pourquoi ?
le site a été crée dans le but de proposer un moyen simple d'allez voir le menu du jour, il est maintenant en ligne depuis un an, et l'equipe n'est plus en mesuire de le maintenir. l'equipe se constituer de menbre qui ne son plus la cette année, il est donc imposible de maintenir le site en place. l'ancien site reste disponible ici, meme si il n'est pas fonctionnel.
la fin ?
ce n'est pas par ce que le site ne peut plus etre maintenu qu'il le sera toujour ! le site est un projet open source il est donc posible pour nimporte qui de reprendre le projet et de le maintenir. il fautdra cepandant avoir quelque conaissance: metriser react et github. lancienne equipe est disponible pour vous donée de l'aide a cette adresse mail: contact@menuvox.fr il faudra cepandant prevoir un moyen d'heberger le site et de maintenir le nom de domaine
